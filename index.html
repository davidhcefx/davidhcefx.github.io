Works

Identicon Generator (Github-like)

An Identicon is an icon that contains a hash value representing part of your identity. Although it is not known how Github generates each user's Identicon, it's not too hard to make one. :-)

Modern Nano Keybindings

This is by far (2020) the most comprehensive keybinding file for nano! It has lots of mainstream keystrokes (^C, ^Z, ^Q) as well as vim-like support (^F, ^G). Another plus is that it is still up-to-date with the latest nano :)

Windows MessageBox for Cmd

This is a command-line (CLI) tool, which is in fact just a Cpp wrapper that utilizes the MessageBox WinAPI. It brought people the convenience of creating a pop-up dialog from cmd, for example: messagebox "Reminder" "Please turn off the lights" "ok" "info".

GNU nano - nosearchwrap

While performing textual search, many editors, for instance Sublime, support toggling the option Wrap, which is to wrap past end-of-file and proceed searching. It will be nice if Nano can have this too!

GNU nano - Bsp-enabled

Ctrl + Backspace is a modern keybinding that has already been widely supported in most browsers, text editors, search bars, file explorers, etc. However, the nanorc file has very limited adjustabily on keybindings, so it's time for us to compile it ourselves!

Search Keyword in Files

Searching for cross references within a huge code base can be cumbersome. This is a simple and efficient tool to perform file-content search within a subdirectory.

Trash - A rm replacement on Cygwin

Ever worried about accidentally deleting something and not being able to recover it? Here are the ways to replace the native rm with a safer version of deleting files via moving it to the Trash bin.
